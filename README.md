# Odoo 19 Repository

This repository contains vendored Odoo 19.0 source code and design themes for Odoo.sh deployment.

## Structure

- `/odoo` - Official Odoo 19.0 core source (vendored)
- `/addons/design-themes` - Official Odoo design themes for version 19.0 (vendored)

## Branches

- `dev` - Development branch
- `staging` - Staging branch
- `production` - Production branch

## Notes

- All upstream code is vendored (copied) at specific versions
- No git submodules are used
- This repository is not a fork of Odoo
